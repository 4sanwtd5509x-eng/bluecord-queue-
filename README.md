## Installation

### Method 1: Direct GitHub URL
1. Open Bluecord
2. Go to Extensions
3. Tap "Add Extension"
4. Enter: `https://github.com/yourusername/bluecord-message-queue`
5. Tap "Install"

### Method 2: Manual Installation
1. Download this repository
2. Place the folder in: `/sdcard/Bluecord/extensions/`
3. Restart Bluecord
4. Enable the extension in settings

## Usage
1. **Enable the extension** in Bluecord settings
2. **Type a message** in any channel
3. **Instead of sending immediately**, it will be added to queue
4. **Messages will be sent automatically** with the configured delay
5. **Tap the queue button** (â±ï¸) in bottom-right to view/manage queue

## Settings
- **Delay between messages**: 500ms to 10s (default: 2s)
- **Max queue size**: 5 to 200 messages (default: 50)
- **Auto-send**: Automatically send queued messages
- **Show notifications**: Display toast notifications
- **Queue on Enter**: Queue when pressing Enter (vs Shift+Enter for immediate)
